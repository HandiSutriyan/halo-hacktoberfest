Bidang Lomba rancang Bangun WEB
ITARSI Fair 2020

Sebuah lomba yang diadakan untuk menguji kemampuan Taruna/i Instrumentasi STMKG dalam pengembangan website.

Tentang Lomba

ITARSI Fair tahun 2020 bidang lomba Web Design akan memperlombakan kompetensi meliputi pengetahuan, pemahaman, dan keterampilan di bidang web design dan pemrograman web. Bidang lomba ini akan dikerjakan secara berkelompok Lomba yang dilaksanakan merupakan satu kesatuan yang utuh, mengacu pada kemampuan teori dan praktek khususnya pada kemampuan design dan kemampuan dalam merancang sistem web.

Tahapan lomba

Submit Karya

Pada tahap ini peserta mengupload karya yang sesuai dengan soal yang diberikan kepada panitia melalui email. Karya dikirim dalam bentuk zip berisi desain mockup laman web yang dibuat, dan semua source file karyanya.

Penyisihan

Tahap penyisihan, dilakukan dengan dua cara yaitu penjurian dan voting. Hasil voting dan penjurian kemudian digabungkan dengan bobot yang sudah ditentukan panitia.

Presentasi

Presenstasi hanya diikuti oleh 5 peserta yang lolos tahap penyisihan. Presentasi dilakukan secara langsung didepan juri dan peserta juga harus mendemonstrasikan website karyanya didepan juri.

Aspek Penilaian

Desain

Kemampuan untuk mendesain sebuah web yang responsive, dan Keterampilan yang diperlukan untuk merancang web dan memanfaatkan komposisi warna dan tipografi. Prinsip-prinsip dan teknik untuk menggunakan grafis untuk digunakan dalam layout website dan logo

Layouting Desain

Bagaimana mengidentifikasi aturan CSS yang tepat untuk mendapatkan hasil yang diinginkan dan menerapkan desain dalam layout website.

Client side programming

Bagaimana mengintegrasikan JavaScript dengan template HTML yang telah dibuat.

Innovative Content

Mengembangkan mengembangkan konten atau fitur yang diperlukan dalam suatu web, membuat rancanngan konten atau fitur yang inovatif dan sesuai dengan target pengguna web. dan membuat rancangan konten atau fitur yang menarik dan mampu menambah nilai guna suatu web.

Skill yang diperlukan

UI/UX Design

UI dan UX adalah singakatan dari User Interface dan User experience yakni merupakan sebuah tampilan visual dalam sebuah aplikasi atau alat pemasaran digital dalam bentuk website yang dapat meningkatkan brand yang dimiliki oleh bisnis atau perusahaan yang dimiliki.

HTML

HTML adalah kependekan dari Hyper Text Markup Language, yaitu sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web. Si HTML inilah yang menyusun sebuah halaman web menjadi sebagaimana yang kita lihat melalui browser (penjelajah internet).

CSS

Secara istilah, CSS adalah bahasa penulisan yang digunakan untuk mendeskripsikan penampilan sebuah dokumen markup. CSS selalu digunakan bersamaan dengan HTML, meskipun kita dapat juga menggunakannya untuk mengatur style dokumen markup lain seperti XML, SVG dan XUL.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat sebuah halaman web yang dinamis dan interaktif. Inilah yang memberi sebuah kehidupan pada halaman – elemen interaktif dan animasi yang dapat menarik perhatian dan melibatkan pengunjung website.
